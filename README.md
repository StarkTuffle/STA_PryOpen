# StarkTruffle Additions - Pry Open

> A highly configurable crowbar prying mod for Project Zomboid

---

## Overview

This mod lets you use a crowbar to pry open a variety of locked objects. Success and punishment rates are determined by skills, traits, and other factors. Fully customizable within sandbox settings to play the way that you want to. 

---

## Features

- Pry open locked doors, windows, and vehicles using a crowbar. (Compatibile with modded crowbars)
- Skill-based success rates influenced by Strength, skills, and traits.
- Dynamic consequences like hand injuries, alarm triggers, and tool wear.
- Supports both context menu and radial interactions for smooth gameplay.
- Fully customizable through sandbox settings and mod options.
